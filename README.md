# 🛒 Velora

Velora is a beginner-friendly e-commerce web application built using **HTML, CSS, and JavaScript**. It demonstrates the core concepts of front-end web development, including dynamic product rendering, shopping cart management, and local storage.

## ✨ Features

* Browse products in a clean and responsive interface
* Add products to the shopping cart
* Increase or decrease product quantity
* Remove items from the cart
* Automatic total price calculation
* Cart data saved using Local Storage
* Responsive design for different screen sizes

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 🚀 Getting Started

1. Clone the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.

No additional setup or installation is required.
